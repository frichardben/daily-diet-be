- [Git](https://git-scm.com/downloads)
- [Node](https://nodejs.org/en/download/) _(version 18 or greater)_.

### :anchor: Installing

```bash
$ git clone https://github.com/frichardben/daily-diet-be
# Go into the repository
$ cd daily-diet-be
# Install dependencies
$ npm i
# Run the project
$ npm run dev
```
